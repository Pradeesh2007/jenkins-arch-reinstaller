# Jenkins Arch Reinstaller

A professional Bash script to **completely remove and reinstall Jenkins on Arch Linux** in a clean and automated way.

This script removes all Jenkins data, users, and services, then performs a fresh installation and automatically retrieves the **initial admin password**.

Perfect for developers who want to quickly reset Jenkins without manual cleanup.

---

## Features

✔ Full Jenkins cleanup  
✔ Removes Jenkins packages, users, and data  
✔ Fresh Jenkins installation  
✔ Automatic service restart  
✔ Automatic admin password retrieval  
✔ Internet connection check  
✔ Colored terminal output  
✔ Ctrl+C safe exit handling  

---

## Supported System

Arch Linux  
BlackArch Linux
